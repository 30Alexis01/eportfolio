Vue d’ensemble

Cette page présente mon parcours d’ingénieur à l’ENSEEIHT, en mettant l’accent sur les compétences développées, les projets réalisés et les productions concrètes illustrant ma progression en génie logiciel.

  • Formation / spécialité : Diplôme d’ingénieur à l’ENSEEIHT (Toulouse INP) — parcours Génie Logiciel
  • Thématiques principales : Génie logiciel, programmation, algorithmique, systèmes, ingénierie dirigée par les modèles
  • Outils utilisés : Git/GitHub, Linux, OCaml, Java, Python, C, Eclipse, VS Code, JUnit

Cours

Projets

Cette section met en avant les projets d’ingénierie auxquels j’ai participé, individuellement ou en équipe.

Prédiction des résultats de matchs de football

  • Objectif : Prédire les résultats de matchs à l’aide de l’analyse de données et du machine learning
  • Rôle : Analyse des données, choix et implémentation des modèles
  • Méthodes / outils : Python, clustering, k-means, DecisionTreeRegressor
  • Résultats : Modèles de prédiction fonctionnels et analyses comparatives
  • Livrables : Rapport de projet et code
  • Pistes d’amélioration : Explorer d’autres modèles et améliorer la qualité des données

Analyse et prédiction des médailles olympiques

  • Objectif : Analyser les facteurs influençant les résultats olympiques et prédire les performances futures
  • Rôle : Analyse de données, modélisation et conception de simulations
  • Méthodes / outils : Analyse de données, simulations, raisonnement statistique
  • Résultats : Prédictions pour les Jeux Olympiques de 2028
  • Livrables : Rapport analytique et modèles de prédiction
  • Pistes d’amélioration : Intégrer des facteurs de performance et de contexte supplémentaires

Site de courtage en ligne (projet collaboratif)

  • Objectif : Concevoir et développer une plateforme web collaborative de courtage en ligne
  • Rôle : Développement logiciel au sein d’une équipe
  • Méthodes / outils : Technologies web, développement collaboratif
  • Résultats : Application en cours de finalisation
  • Livrables : Code source et prototype web
  • Pistes d’amélioration : Finaliser les fonctionnalités et renforcer la robustesse
